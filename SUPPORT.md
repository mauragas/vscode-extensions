# Support

## Questions and usage help

If you have a question about setup, usage, or packaging, open an issue in this repository:

- <https://github.com/mauragas/vscode-extensions/issues>

## Bug reports

When reporting a bug, please include:

- the extension name and version
- your VS Code version
- your operating system
- clear reproduction steps
- screenshots or logs when helpful

## Feature requests

Feature requests are welcome. Please describe the problem you are trying to solve and how you expect the extension to behave.

## Responsible reporting

Do not include secrets, private repository data, or access tokens in public issues.
